# Gerador Dicas

Este projeto é um gerador de dicas de estudo desenvolvido para auxiliar estudantes a aprimorarem seus hábitos e estratégias de aprendizagem. Ajudando o usuário a estudar de forma mais eficiente e equilibrada.
O objetivo do gerador é oferecer orientação rápida e acessível para quem deseja melhorar seu desempenho acadêmico, manter a motivação e criar uma rotina de estudos mais produtiva.

# Funcionalidades

  - Clique no Botão e gere uma dica de estudo
  - Layout Responsivo



    
# Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Git e Github

## Como Rodar

        1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
		
        2. Extraia a pasta
		
        3. Abra o arquivo index.html no navegador

## Como Clonar

No terminal do VS Code digite:

	git clone https://github.com/Fellipe-07/gerador-dicas.git
	
No GitHub Desktop:

	No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Fellipe-07/gerador-dicas.git
